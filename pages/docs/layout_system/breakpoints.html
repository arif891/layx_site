Breakpoints in LayX

LayX uses a flexible and responsive grid system with predefined breakpoints that adapt layouts to various screen sizes. These breakpoints allow you to design responsive websites effortlessly, ensuring optimal layouts across devices.

Available Breakpoints
  • sm: 576px
  • md: 768px
  • lg: 992px
  • xl: 1200px
  • xxl: 1400px
  • wd: 16 / 9
  • uwd: 21 / 9
  • swd: 32 / 9

LayX’s breakpoints allow you to define different styles and layout structures based on screen width, making it simple to achieve fully responsive designs.

Usage Example

Here’s an example of how you can use breakpoints in your CSS:

/* Define breakpoints for different screen sizes */ :root { --breakpoint-xs: 0px; /* Extra small devices (phones, less than 576px) */ --breakpoint-sm: 576px; /* Small devices (tablets) */ --breakpoint-md: 768px; /* Medium devices (desktops) */ --breakpoint-lg: 992px; /* Large devices (larger desktops) */ --breakpoint-xl: 1200px; /* Extra large devices (wider desktops) */ --breakpoint-xxl: 1400px; /* Extra extra large (very wide desktops) */ --breakpoint-xxxl: 1800px; /* Extra extra extra large (ultra-wide screens) */ /* Aspect ratio breakpoints */ --breakpoint-wd: 16 / 9; /* Standard widescreen (16:9 aspect ratio) */ --breakpoint-uwd: 21 / 9; /* Ultrawide screens (21:9 aspect ratio) */ --breakpoint-suwd: 32 / 9; /* Super ultrawide screens (32:9 aspect ratio) */ } /* Media Queries for larger screens */ /* (sm) Small devices (landscape phones, 576px and up) */ @media (min-width: 576px) { /* Add styles for small devices here */ } /* (md) Medium devices (tablets, 768px and up) */ @media (min-width: 768px) { /* Add styles for medium devices here */ } /* (lg) Large devices (desktops, 992px and up) */ @media (min-width: 992px) { /* Add styles for large devices here */ } /* (xl) Extra large devices (1200px and up) */ @media (min-width: 1200px) { /* Add styles for extra-large devices here */ } /* (xxl) Extra extra large devices (1400px and up) */ @media (min-width: 1400px) { /* Add styles for very wide screens here */ } /* (xxxl) Ultra-wide screens (1800px and up) */ @media (min-width: 1800px) { /* Add styles for ultra-wide screens here */ } /* Aspect ratio-based media queries */ /* Widescreen displays (16:9 and wider, 1800px and up) */ @media (min-aspect-ratio: 16 / 9) and (min-width: 1800px) { /* Styles for wide screens here */ } /* Ultrawide displays (21:9 and wider, 2000px and up) */ @media (min-aspect-ratio: 21 / 9) and (min-width: 2000px) { /* Styles for ultrawide screens here */ } /* Super ultrawide displays (32:9 and wider, 3000px and up) */ @media (min-aspect-ratio: 32 / 9) and (min-width: 3000px) { /* Styles for super ultrawide screens here */ } /* Media Queries for smaller screens */ /* (xs) Extra small devices (below 576px) */ @media (max-width: 575px) { /* Add styles for extra small devices here */ } /* (sm) Small devices (below 768px) */ @media (max-width: 767px) { /* Add styles for small devices here */ } /* (md) Medium devices (below 992px) */ @media (max-width: 991px) { /* Add styles for medium devices here */ } /* (lg) Large devices (below 1200px) */ @media (max-width: 1199px) { /* Add styles for large devices here */ } /* (xl) Extra large devices (below 1400px) */ @media (max-width: 1399px) { /* Add styles for extra large devices here */ } /* (xxl) Extra extra large devices (below 1800px) */ @media (max-width: 1799px) { /* Add styles for extra extra large devices here */ } /* Narrower than widescreen (below 16:9 aspect ratio, 1800px) */ @media (max-aspect-ratio: 16 / 9) and (max-width: 1799px) { /* Styles for narrower than wide screens here */ } /* Narrower than ultrawide (below 21:9 aspect ratio, 2000px) */ @media (max-aspect-ratio: 21 / 9) and (max-width: 1999px) { /* Styles for narrower than ultrawide screens here */ } /* Narrower than super ultrawide (below 32:9 aspect ratio, 3000px) */ @media (max-aspect-ratio: 32 / 9) and (max-width: 2999px) { /* Styles for narrower than super ultrawide screens here */ }

By customizing breakpoints, you can create adaptable layouts that look great on all screen sizes.
