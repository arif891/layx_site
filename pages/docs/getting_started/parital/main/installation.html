Installation

To install Layx on your system:

  1. Download the Layx setup file from our download page.
  2. Extract the contents of the ZIP file.
  3. Open a terminal or command prompt and navigate to the extracted directory.
  4. Run the following command:
    5. layx install Note: On Linux and macOS, before installation, you need to run chmod +x ./layx.sh to make it executable. Once installed, all commands remain the same.
