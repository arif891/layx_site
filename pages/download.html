Installation

To install Layx on your system:

  1. Extract the contents of the ZIP file.
  2. Open a terminal or command prompt and navigate to the extracted directory.
  3. Run the following command:
    4. layx install Note: On Linux and macOS, before installation, you need to run chmod +x ./layx.sh to make it executable.

For full installation instructions, visit the Installation Guide.

Note: The macOS version is currently untested and may not function as expected. If you encounter any issues, please report them by creating an issue on our GitHub repository.
